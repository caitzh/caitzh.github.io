Hexo yilia主题

Hexo可以很方便地创建个人博客，需要用到nodejs和github，参考教程如下：

Hexo安装和配置

之后我更换了yilia主题，但是出现了一些小问题，解决办法如下：

1. 头像无法加载

在 _config.yml 设置了 avatar 的 url 后，启动服务器，头像却无法显示，在浏览器按F12后发现，图片的url前自动加上了一个null，解决办法是 root 值不要放空，设为 “” 即可，如下图：

同时还需要把动画效果关闭, 若想不关闭动画效果，请看下面第三个问题”3.下拉菜单不显示”：


2. 代码块出现滑动条

把themes\yilia\source\css\_partial\highlight.styl 里的68行 height: 15px 注释掉即可


3. 下拉菜单不显示

根据此链接修改对应文件即可，此方法还可以解决上面第一个问题的头像动画效果

4. 如何把房子的下拉菜单改成中文

把 hexo 的配置文件_config.yml里的 language 设为 zh-Hans


5. 点击所有文章出错

在本地可以打开，但是 deploy 到 github 上之后便无法打开，总是自动从 https 重定向到 http, 解决方法是把 _config.yml 里的 menu/所有文章 的最后加上一个 /

# 原来的
    所有文章: /archives
# 修改后
    所有文章: /archives/
